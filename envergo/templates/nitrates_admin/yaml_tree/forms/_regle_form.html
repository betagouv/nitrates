Aide saisie des bornes (du / au)
  • Date fixe : 15/12
  • Événement : date_semis_couvert
  • Événement décalé : date_semis_couvert+4semaines, date_destruction_prevue-20jours
Unités : jours | semaines | mois. Signe : + ou - suivi d'un entier ≥ 1.
{# ─── Aide saisie des conditions (optionnelles) ─────────────── #}
Aide saisie des conditions (optionnelles)

Format : <input_id> <opérateur> <JJ/MM>

  • input_id : un input requis ci-dessus (type date)
  • opérateurs : < <= > >= == !=
  • date : JJ/MM strict
Exemples : date_destruction_couvert < 05/12, date_semis_couvert >= 01/09.
Note : la comparaison suit l'ordre de l'année agricole (juillet → juin). 01/08 < 05/12 est vrai.
